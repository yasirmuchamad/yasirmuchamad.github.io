About Me

Saya adalah lulusan Software Enginer di SMK Negeri 4 Kendal, di studi tingkat pertama saya mempelajari bahasa pemrograman Pascal, Borland Delphi dan juga Web Design HTML menggunakan Dreamweaver. Pada tingkat kedua saya mulai berkenalan dengan bahasa pemrograman PHP. Kemudian meski pada tingkat akhir bahasa pemrograman Java yang menjadi fokus studi, namun pihak penguji meminta tugas akhir dibuat menggunakan gabungan PHP, HTML dan CSS. Dan jadilah Sistem Informasi Perpustakaan yang mengakhiri pendidikan menegah saya.

Pertama kali meniti karir di bidang IT yaitu saat saya diterima di perusahaan perbankan diwilayah kota semarang. dengan tanggung jawab maintenance Hardware, Software dan Jaringan di seluruh kantor, baik itu pusat, cabang dan juga kantor-kantor Kas perusahaan. Disana saya Kemudian mempelajari Visual Basic, Visual FoxPro, Java Desktop OOP,dan juga Python. Kemudian saya juga mulai mempelajari teknologi-teknologi Framework antara lain Code Igniter dan Laravel pada PHP, Springboot pada Java, Django dan Flask pada Python. Dan saat ini saya sedang mempelajari Go-Lang.
